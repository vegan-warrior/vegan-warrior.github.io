# [vegan-warrior.github.io](https://vegan-warrior.github.io)

## Install

1. Clone
1. `pip install Jinja2 markdown`

## How to build

1. Change html files in `templates/` or py files in `{language}/`
1. Run `python build.py`
1. Check, commit, push
